# 硕士论文前期准备检查清单

## 选题
- [ ] 题目不超过两个核心变量或一个核心机制
- [ ] 研究对象具体
- [ ] 时间、平台、人群边界清楚

## 文献
- [ ] 至少整理30篇核心文献
- [ ] 中英文文献都有
- [ ] 记录了理论、方法和局限

## 问题
- [ ] 有1个主研究问题
- [ ] 有2-4个子问题
- [ ] 每个问题都能用数据回答

## 方法
- [ ] 方法与问题匹配
- [ ] 样本可获得
- [ ] 工具/问卷/编码表可设计

## 伦理
- [ ] 访谈和问卷有知情同意
- [ ] 平台数据不侵犯隐私
- [ ] AI使用有记录和边界
