# 开题答辩反方审查 Prompt

请作为严厉但建设性的传播学硕士开题答辩委员，审查以下研究设计。

研究题目：
【粘贴题目】

研究设计摘要：
【粘贴开题报告摘要/目录/研究问题/方法】

请从以下角度审查：
1. 题目是否过宽；
2. 核心概念是否清楚；
3. 文献综述是否可能重复；
4. 研究问题是否能回答；
5. 方法是否匹配问题；
6. 样本和数据是否可获得；
7. 理论贡献是否成立；
8. 伦理风险；
9. 最可能被老师追问的10个问题；
10. 修改优先级。

输出要求：
- 先给总分，0-100；
- 再给必须修改、建议修改、可以保留三类意见；
- 不要空泛批评，要给具体改法。
