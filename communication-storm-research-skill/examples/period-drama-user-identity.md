# 示例3：年代剧用户认同建构

## 初始题目
共情视域下年代剧用户认同建构研究——以《小巷人家》为例

## STORM 式视角

| 视角 | 关键问题 |
|---|---|
| 影视传播 | 叙事结构如何引发观看者共情？ |
| 情感传播 | 观众的共情对象是人物、家庭还是时代？ |
| 文化研究 | 年代剧如何建构集体记忆和怀旧情绪？ |
| 代际传播 | 年轻观众如何理解父辈经验？ |
| 受众研究 | 弹幕、评论和社交媒体讨论如何放大认同？ |
| 方法论 | 应做文本分析、评论分析还是访谈？ |

## 方法建议
文本分析 + 社交媒体评论内容分析 + 少量访谈。核心是把“共情”拆成可分析的叙事触发点和用户表达证据。
