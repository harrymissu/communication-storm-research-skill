# 示例2：小红书考研类博主粉丝社群

## 初始题目
社群认同对成员粘性的影响研究——以小红书考研类博主的粉丝社群为例

## STORM 式视角

| 视角 | 关键问题 |
|---|---|
| 社群传播 | 粉丝社群如何形成边界和规范？ |
| 平台传播 | 小红书推荐机制如何强化社群连接？ |
| 教育传播 | 考研内容如何提供信息与情绪支持？ |
| 社会认同 | 成员如何把个人目标绑定到群体身份？ |
| 粉丝文化 | 考研博主是否形成陪伴式权威？ |
| 方法论 | 粘性用互动频率、停留时间还是持续关注衡量？ |

## 可执行研究问题
RQ1：考研类博主粉丝社群中的社群认同由哪些维度构成？  
RQ2：社群认同如何影响成员持续关注、互动和内容转发？  
RQ3：情感支持是否在社群认同与成员粘性之间发挥中介作用？
