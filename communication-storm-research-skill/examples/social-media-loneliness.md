# 示例1：社交媒体使用与大学生孤独感

## 初始题目
社交媒体使用与大学生孤独感的相关性研究

## STORM 式视角

| 视角 | 关键问题 |
|---|---|
| 媒介效果 | 使用强度是否影响孤独感？ |
| 社会心理学 | 社会比较是否中介这种关系？ |
| 平台机制 | 被动浏览与主动互动是否不同？ |
| 青年研究 | 大学生转型期的关系压力如何介入？ |
| 方法论 | 孤独感如何测量？使用时长是否可靠？ |
| 反方解释 | 本来孤独的人是否更依赖社交媒体？ |

## 可执行研究问题
RQ1：不同类型的社交媒体使用行为与大学生孤独感是否相关？  
RQ2：社会比较是否在社交媒体使用与孤独感之间发挥中介作用？  
RQ3：线上社会支持是否能够削弱孤独感？

## 方法建议
问卷调查 + 中介/调节模型。注意区分主动互动、被动浏览、内容发布和私信沟通。
