# 研究方法设计手册（Method Design Deep）

> 编纂：**西南交通大学 传播系 黄昕恺，Ph.D in Communication from University of Utah**
>
> 把"一方法一行"的表，扩成能过开题答辩的方法工程。每种方法按统一结构：`适用问题 / 抽样或取材 / 工具或编码 / 信效度或可信度标准 / 常见硬伤与答辩追问 / 硕士可行性`。配合 `docs/deep-research-engine.md` 与 `docs/hot-topics-2024-2026.md` 使用。

## 一、定量方法

### 1. 问卷调查（Survey）
- **适用问题**：变量关系、机制模型（中介/调节）、群体差异。
- **抽样**：概率抽样优先；便利样本须说明局限。样本量按统计功效（power ≥ .80）与模型复杂度估算；SEM 经验上每个估计参数 ≥ 5–10 个样本。
- **工具**：**优先借用已验证量表**并做翻译—回译与本土化；自编量表须报告探索性/验证性因子分析。
- **信效度**：信度（Cronbach's α ≥ .70，组合信度 CR）、效度（内容、聚合 AVE ≥ .50、区分）。**共同方法偏差（CMB）**须做程序性控制（匿名、题项分隔）+ 统计性检验（Harman 单因子、共同方法因子）。
- **常见硬伤/追问**：反向因果（"越孤独越用 vs 越用越孤独"）；只写"问卷调查"不交代变量与控制变量。
- **硕士可行性**：高（在线问卷可行），但样本代表性是评审重点。

### 2. 内容分析（Content Analysis）
- **适用问题**：文本/评论/视频中"有什么模式、占多少比例"。
- **抽样**：明确**分析单元（unit of analysis）**与抽样框（时间窗、平台、关键词、抓取规则）。
- **工具**：可操作化的**编码表（codebook）**，每个类目有定义、示例、纳排标准。
- **信度**：**编码员间信度（intercoder reliability）**须报告系数与阈值——名义变量用 **Krippendorff's α 或 Cohen's κ**（一般 α/κ ≥ .80 可接受，.667–.80 谨慎）；先做预测试再正式编码。
- **常见硬伤/追问**：编码类目不互斥/不穷尽；样本随意；只报百分比不报信度。
- **硕士可行性**：高，但人工编码工作量需提前估算（双编码）。

### 3. 实验（Experiment）
- **适用问题**：因果影响、刺激材料效果。
- **设计**：组间/组内/混合；**随机分配**；设**操纵检验（manipulation check）**与**注意力检验（attention check）**。
- **效度**：内部效度（控制混淆）、外部/生态效度（材料与情境的真实性）、统计结论效度。
- **常见硬伤/追问**：生态效度不足；操纵不成功却不自知；一次性测量混淆即时与持久效应。
- **硕士可行性**：中（线上实验可行，刺激材料制作与样本是成本）。

### 4. 计算传播 / 数据科学（Computational）
- **适用问题**：大规模文本主题、情感、传播网络结构。
- **取材**：注意平台条款与法律伦理边界；记录抓取时间与口径。
- **工具**：主题模型（LDA/BERTopic）、情感分析、社会网络分析；机器编码须与**人工编码做一致性验证**。
- **效度**：模型可解释性、与人工编码的一致性、稳健性检验。
- **硕士可行性**：取决于数据可得性与算力；门槛较高时优先与人工编码混合。

## 二、定性方法

### 1. 深度访谈（In-depth Interview）
- **适用问题**：体验、意义、动机、身份建构。
- **抽样**：目的性/理论/滚雪球抽样；以**理论饱和**判断样本量（新访谈不再带来新主题）。
- **工具**：半结构提纲分层（主问题—追问—探查）。
- **可信度（trustworthiness）**：可信性（成员校验、三角验证）、可迁移性（厚描）、可靠性、可确证性；**研究者反身性**须交代。
- **常见硬伤/追问**：样本少却过度推论；把受访者原话当结论而无分析层。
- **硕士可行性**：高，但转写与编码耗时。

### 2. 田野 / 参与观察（Ethnography / Participant Observation）
- **适用问题**：社区实践、互动情境、文化意义（如数字游民社区）。
- **取材**：进入许可与角色协商；规范**田野笔记**；**研究者位置（positionality）**透明。
- **可信度**：长期投入、厚描（thick description）、从描述到阐释的跨越。
- **常见硬伤/追问**：只描述不阐释；忽略自身在场对场域的影响。
- **硕士可行性**：中（需时间与进入条件）。

### 3. 文本 / 话语分析（Textual / Discourse Analysis）
- **适用问题**：文本如何承载意义、权力、身份、记忆。
- **关键**：**先选定分析框架**——框架分析（Entman）、批判话语分析（Fairclough）、叙事分析、符号学、扎根理论——不同框架"什么算证据、怎样算分析（而非复述）"完全不同。
- **可信度**：分析步骤透明、样本与语境清楚、避免印象式解读。
- **常见硬伤/追问**：把"赏析/复述"当分析；没说清用哪种框架。
- **硕士可行性**：高。

### 4. 扎根理论（Grounded Theory）
- **流程**：开放编码 → 主轴编码 → 选择性编码；持续比较；写分析备忘录。
- **适用**：缺乏现成理论、需从数据生成理论时。
- **硕士可行性**：中（编码强度高，易半途而废）。

## 三、混合方法（Mixed Methods）
- **三种经典设计**：解释性序列（先量后质，用质性解释量化结果）、探索性序列（先质后量，用质性发展量表/假设）、聚合并行（量质同时，结果互证）。
- **整合**：明确**整合点（point of integration）**与**联合展示（joint display）**。
- **硕士可行性**：警告——工作量常超出单人一年期，须先评估再上。

## 四、方法—问题匹配诊断器

| 研究问题类型 | 推荐方法 | 不推荐 | 最常被答辩否掉的点 |
|---|---|---|---|
| 变量关系/机制 | 问卷 + 中介调节/SEM | 纯描述性问卷 | 反向因果、变量未操作化 |
| 因果影响 | 实验 | 横断面问卷下因果结论 | 生态效度、操纵失败 |
| 意义/体验/认同 | 深度访谈 | 大样本问卷 | 样本少却过度推论 |
| 内容分布/模式 | 内容分析 | 主观举例 | 无信度、抽样随意 |
| 社区/情境过程 | 田野 + 访谈 | 仅问卷 | 只描述不阐释 |
| 复杂机制需互证 | 混合方法 | 单一方法硬撑 | 整合点不清、工作量爆炸 |

> 口诀：看关系→问卷；看意义→访谈；看内容分布→内容分析；看因果→实验；看复杂机制→混合方法。**但永远先问：我的研究问题是哪一类？**

## 五、伦理底线（所有方法通用）
- 访谈/问卷/实验须知情同意；老年、患者、未成年等脆弱群体从严。
- 平台数据遵守平台条款与隐私，不发布可识别个人的原始数据。
- AI 使用须留痕、可说明；不以 AI 代替真实数据采集与分析。
